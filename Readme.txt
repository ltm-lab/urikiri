앱 : 우리끼리
Target SDK : Android 11 (api 30)

기능 설명
 - 카카오, 페이스북을 통한 소셜 로그인
 - 임시 토큰을 발급받아 비회원 로그인 가능(앱 삭제시 데이터 초기화)
 - ID를 통한 친구 추가 및 삭제
 - 원하는 이름 및 설명으로 그룹 생성
 - 위치를 공유하고 싶은 그룹만 선택하여 공유 On/Off
 - 카카오맵 API를 이용한 맵뷰 (AVD 실행환경 작동 X)
 - 자신의 위치가 맵뷰에 마커로 표시
 - 위치 공유 그룹에 참가한 멤버 확인 가능

*몇몇 기능은 더미데이터를 통해 작동을 확인하였고, 현재는 실 서비스에 맞게 앱을 수정하여 데이터가 없을 수 있습니다.
